## Effective Layer Definition

### Humans as Tension-Maintaining Systems

在張力宇宙的有效層中，一個活著的人被定義為：

> **能長時間維持張力閉環穩定的系統**

形式化表達（有效層）：

```

∇ · J_T ≈ 0

```

意義在於：

- 張力可流動  
- 張力可補償  
- 張力可修復  
- 不會在局部無限累積或崩潰  

這就是「活得久」在結構上的定義。

## Why Some Smokers and Drinkers Live Long

在有效層中，抽菸與喝酒不是主導變數，  
而是 **外部擾動項**：

```

T_ij(total) = T_ij(internal) + ΔT_ij(external)

```

真正決定結果的是：

> **系統是否能吸收這些擾動而不產生相變**

某些個體具有：

- 深張力井  
- 快速修復回路  
- 分散的張力流  
- 無奇異點集中  

因此外在破壞行為 **未觸發結構崩潰**。

---

## Why Healthy Lifestyles Sometimes Fail

「養生行為 ≠ 張力穩定」。

部分系統在有效層呈現：

```

∇ · J_T ≠ 0

```

特徵包括：

- 張力卡滯於特定節點  
- 補償路徑受阻  
- 微擾無法被消化  
- 張力局部累積  

結果是外表健康，內部卻持續耗損。  
這代表系統已接近 **臨界狀態**。

---

## Why the Same Drug Works for Some but Not Others

在有效層中，藥物不是「病因解決器」，而是：

```

ΔT_ij(drug)

```

其生效條件為：

```

∃ coupling such that ΔT_ij(drug) → tension redistribution

```

若張力場：

- 已重構  
- 張力井位置不同  
- 補償路徑繞過藥物作用點  

則藥物成為 **無法耦合的擾動**。

這不是藥無效，而是 **對該張力結構無效**。

## What Personalized Medicine Lacks

缺的不是更多基因、更多 AI、更多數據，  
而是這個問題的答案：

```

Is the individual's tension loop currently stable?

```

若此變數未進模型：

- 相同藥物結果必然分歧  
- 相同行為後果必然相反  
- 統一指南必然無法普適  

這不是例外，而是結構必然。
